App Name: TeamFlow - Team Task Manager
Live URL: https://team-task-manager-production-5f56.up.railway.app
GitHub: https://github.com/hitenkatyal/team-task-manager

Features:
- JWT Authentication (Signup/Login)
- Role-based access (Admin/Member)
- Project & team management
- Task creation, assignment & status tracking
- Kanban board per project
- Dashboard with stats (total, overdue, done)
- REST API with SQLite database
- Deployed on Railway

Tech Stack:
- Backend: Node.js, Express.js
- Database: SQLite (@libsql/client)
- Frontend: React 18, React Router
- Auth: JWT + bcrypt
- Deployment: Railway
